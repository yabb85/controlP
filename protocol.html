Protocol definition¶

This page explain the protocol used by pioneer N-50A

The communication between your application and the player is a socket opened on port 8102. If you use a NAS to store your files you must use http to retrieve the cover of songs. The protocol have two command type, the first is the command without response and the second is command with response.
